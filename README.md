# Innoworq Workspace

Official web application repository for Innoworq Infotech Pvt. Ltd.

## Project Structure

- `client/`: React + Vite frontend application with WebGL / Three.js interactive components.
- `server/`: Backend service API.

## Getting Started

### Installation

Install dependencies for all workspaces:

```bash
npm run install:all
```

### Development

Run both client and server concurrently:

```bash
npm run dev
```

Or run client/server individually:

```bash
npm run dev:client
npm run dev:server
```

### Build

Build the production frontend bundle:

```bash
npm run build
```
